# Bank Customer Churn Analysis

Analyze and predict bank customer churn using machine learning to identify at-risk customers and improve retention.

## Features
- Data preprocessing & EDA
- Machine learning models: Logistic Regression, Decision Trees, Random Forest
- Churn prediction and insights for retention

## Tech
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

## Usage
1. Clone the repo  
2. Install dependencies  
3. Run the scripts or notebooks
